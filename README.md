Repository Status: Structural Preview

The source code provided in this repository is currently a structural preview of the PLGF-NET framework. To protect intellectual property while our manuscript is under peer review, the core proprietary algorithm implementations—including the Sample Prediction Difficulty (SPD) formulations, the Physics-Guided Spatial Attention (PGSA) mechanism, and the fitted Geophysical Model Function (GMF) parameters—have been temporarily redacted (replaced with NotImplementedError).

The complete, fully executable source code, along with the pre-trained weights and data processing scripts, will be publicly released immediately upon the acceptance of the paper.
